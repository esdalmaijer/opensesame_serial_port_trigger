Serial Port Trigger

The Serial Port Trigger plugin allows you to send signals out a serial port. This allows you to, for example, send synchronization send triggers in an EEG experiment.

The following options are available:

  • Value can be a number between 0 and 255

  • Duration should be a positive integer value, 'keypress', or 'mouseclick', to wait respectively for a specified interval in milliseconds, until a key has been pressed, or until a mouse button has been clicked.

  • Port Address (Windows only) should be the the value of the parallel port adress in Windows, e.g. COM1.
